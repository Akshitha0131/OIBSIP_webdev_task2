# Task 3 – Temperature Converter

## Description
This project is a **Temperature Converter Web App** that converts between:  
- Celsius (°C)  
- Fahrenheit (°F)  
- Kelvin (K)  

The interface is styled with a background image and includes input validation.

##  Technologies Used
- HTML5  
- CSS3  
- JavaScript (DOM manipulation, calculations)  

##  How to Run
1. Download or clone this folder.  
2. Open **index.html** in any modern browser (Chrome, Edge, Firefox).  
3. Enter a temperature value, select a unit (°C, °F, or K), and click **Convert**.  
4. The converted values will be displayed instantly.
